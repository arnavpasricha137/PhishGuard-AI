# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Temporary Solution

For development, you can create simple placeholder icons using any image editor or online tool.

Recommended colors:
- Background: #1976d2 (blue)
- Icon: White shield emoji 🛡️ or text "PG"

## Production Icons

For production, create professional icons with:
- Clear shield or security symbol
- PhishGuard branding
- Consistent color scheme
- Transparent background (PNG format)
